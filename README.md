## 🛠️ Технологический стек

- **Backend**: Python 3.12, Flask, SQLAlchemy, Flask-Login, Flask-WTF
- **Frontend**: Bootstrap 5, Jinja2, EasyMDE (Markdown), CSS
- **База данных**: PostgreSQL
- **Дополнительно**: bleach (санитайзер HTML), markdown, Pillow (обработка изображений)

## 📂 Структура проекта
``` LibraryExam/
├── app/
│ ├── init.py # создание приложения, фильтр markdown, инициализация БД
│ ├── models.py # модели SQLAlchemy (Book, User, Review, Genre…)
│ ├── forms.py # WTForms (BookForm, ReviewForm, LoginForm)
│ ├── routes.py # маршруты Flask (книги, рецензии, модерация, аутентификация)
│ ├── utils.py # работа с файлами, Markdown, санитайзер
│ ├── static/
│ │ ├── css/
│ │ │ └── style.css
│ │ └── uploads/ # сохранённые обложки
│ └── templates/ # HTML-шаблоны (base, index, book_detail, review_form…)
├── config.py # конфигурация (URI БД, секретный ключ, папки)
├── run.py # точка входа (с командой init-db)
├── requirements.txt
└── README.md
```
